# PaperLens

PaperLens is a full-stack academic paper exploration platform designed to help students, researchers, and academic enthusiasts discover, organize, and explore research papers more efficiently. The platform combines a modern, user-friendly interface with an intelligent recommendation system that suggests related papers based on their titles, abstracts, and keywords.

Using **TF-IDF** and **Cosine Similarity**, PaperLens analyzes the textual content of research papers to identify meaningful relationships between publications, making it easier for users to discover relevant literature. The application also provides secure authentication, paper management, search and filtering capabilities, and a community-driven review system, all within a responsive glassmorphism-inspired interface.

## Features

* Secure user authentication using JWT and HTTP-only cookies.
* Browse and explore a collection of academic research papers.
* Search and filter papers by keywords, categories, and publication details.
* View comprehensive information for each research paper.
* Receive intelligent paper recommendations using TF-IDF and Cosine Similarity.
* Submit ratings and reviews to share feedback with the community.
* Add, manage, and organize your own research papers.
* Responsive design that provides a seamless experience across desktop, tablet, and mobile devices.
* Modern glassmorphism-inspired user interface designed for an intuitive user experience.
